如何利用hexo与gitpages建站

2022-02-08- 2022-02-14
-教程 -分享 -hexo

1.准备步骤

1.1.下载node.js与git

1.2.注册github账号

1.3.安装Hexo

安装步骤

1.按 win+r 输入 cmd 打开命令行后输入如下代码： 

npm install hexo-cli -g

如果安装失败，可能是没有权限，可以尝试头部加上 sudo 重新执行：sudo npm install hexo-cli -g

2.创建本地博客

新建一个文件夹，名称随意。打开cmd输入：

cd 你新建文件夹所在的地址
hexo init
npm install

至此本地博客便已经建造完成。可以输入hexo s验证一下

3.与远程仓库建立关联

3.1.本地配置用户名和邮箱

git config --global user.name “yourusername”
git config--global user.email “youremail”

3.2.输入命令 git init 把当前目录变成git可以管理的仓库

3.3.输入以下命令生成SSH Key

 ssh-keygen -t rsa -C"你的邮箱"

查看 用户主目录/.ssh/下有两个文件，id_rsa是私钥，id_rsa.pub是公钥

进入该页面SSH Keys，点击new ssh key将私钥复制到key板块内

3.4.测试ssh key是否成功
使用命令“ssh -T git@github.com”，再输入“yes”。

如果出现You’ve successfully authenticated, but GitHub does not provide shell access 。这就表示已成功连上GitHub。

3.5.输入git init与远程仓库建立关联

4.部署

4.1.在 _config.yml 中配置。

deploy:
type: git
repo: 你此前新建的仓库的链接 # 比如https://github.com/YunYouJun/yunyoujun.github.io
branch: master # 默认使用 master 分支
message: Update Hexo Static Content # 你可以自定义此次部署更新的说明

4.2.在命令行中依次输入：

npm install hexo-deployer-git
hexo clean
hexo g
hexo d

等待完成后，打开网址 https://你的名字.github.io 就能看到你的线上网站了。

5.写作

输入以下命令即可新建 xxx.md 文件。

hexo new post xxx

然后你就可以使用markdown来写作了。这是一些基本语法
然后在命令行中进入到你博客所在文件夹，输入：

hexo g
hexo d

即可更新博客
